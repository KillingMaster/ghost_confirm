# GHOST_CONFIRM
## Description
This is an Implementation of the submit Button That Avoids The Cursor While Forms
is invalid or not filled.
## Stack
- React 
- TailwindCSS
## How to run
- Clone the repo
- Run `npm install`
- Run `npm start`
- Open `localhost:3000` in your browser
- Enjoy!
